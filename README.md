# BuyLow OS - 트레이딩 팀 운영 플랫폼

BuyLow Strategy Inc.를 위한 트레이딩 팀 운영, 교육, 리스크 관리 플랫폼입니다.

## ⚠️ 중요 안내

- **매매 추천, 가격 예측, 종목 추천을 제공하지 않습니다**
- 교육 및 정보 제공 목적으로만 사용됩니다
- 모든 투자 결정은 본인 책임입니다

## 🚀 실행 방법

```bash
# 의존성 설치
pip install -r requirements.txt

# 앱 실행
streamlit run Home.py
```

브라우저에서 `http://localhost:8501` 접속

## 📱 주요 기능

### 멤버용

| 기능 | 설명 |
|------|------|
| **온보딩** | 신규 멤버 체크리스트, 등급 시스템 |
| **공지 허브** | 태그별 필터, 검색, 텔레그램 복사 |
| **CS 챗봇** | 키워드 기반 FAQ 검색 |
| **진단 퀴즈** | 8문항 테스트, 맞춤 학습 추천 |
| **과제 제출** | 주제별 과제, 제출 시 콘텐츠 언락 |
| **리스크 체크** | 매매 전 위험 점검, 경고 누적 관리 |
| **해설 모음** | 과제 제출로 언락되는 심화 해설 |
| **심화 문제** | 과제 2회 제출로 언락되는 고급 문제 |

### 운영자용

| 기능 | 설명 |
|------|------|
| **운영자 대시보드** | 한 줄 요약, 막힌 포인트 분석, 과제 지표 |
| **공지 템플릿** | 교육/이벤트/브리핑/주의사항 템플릿 생성 |
| **채점 보조** | 제출물 일괄 확인, 체크리스트 채점 |
| **티켓 관리** | CS 티켓 처리, 알림 문구 생성 |

## 🎓 교육 주제 (PDF 기반)

1. **다이버전스** - RSI/MACD 다이버전스 분석
2. **지지저항** - 지지선과 저항선 분석
3. **SRL** - SRL 지표 설정과 해석
4. **아래꼬리 캔들** - 아래꼬리 캔들 패턴 분석

## 🔓 언락 시스템

과제 제출로 추가 콘텐츠가 열립니다:

- **1회 제출** → 해당 주제 해설 페이지 언락
- **2회 제출** → 해당 주제 심화 문제 언락

## 🛡️ 리스크 경고 누적

- 동일 경고 3회 누적 시 → 미니 리스크 코스 필수
- 손절 미설정, 과도한 레버리지, 과도한 비중, 감정적 상태 등 체크

## 👥 멤버 등급 시스템

행동 기반 등급:
- 🥉 **브론즈** → 🥈 **실버** → 🥇 **골드** → 💎 **플래티넘**

점수 요소:
- 과제 제출 횟수
- 연속 제출 스트릭
- 리스크 규칙 준수
- 자가 해결 비율

## 📋 데모 사용 흐름

### 신규 멤버
1. **온보딩** 페이지에서 닉네임 입력
2. 체크리스트 완료 (멤버십 확인, 학습 순서, 과제 방법)
3. **교육 콘텐츠**에서 학습 순서 확인
4. **과제 제출**로 실습 및 콘텐츠 언락
5. **해설 모음**/**심화 문제**로 심화 학습

### 기존 멤버
1. **공지 허브**에서 새 공지 확인
2. **과제 제출**로 주간 실습
3. **리스크 체크**로 매매 전 점검
4. **CS 챗봇**으로 궁금한 점 해결

### 운영자
1. **운영자 대시보드**에서 현황 파악
2. **채점 보조**에서 과제 채점
3. **관리자**에서 공지 템플릿 생성
4. **티켓 관리**에서 CS 처리
